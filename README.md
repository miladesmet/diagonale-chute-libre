# Diagonale Chute Libre

## Installation

1 - cloner le dépôt :

```bash
git clone https://github.com/miladesmet/diagonale-chute-libre.git
```

2 - installer les différentes librairies utilisées :

```bash
npm install
```

3 - Lancer l'application :

```bash
npm run dev
```

4 - Compiler l'application :

```bash
npm run build
```
